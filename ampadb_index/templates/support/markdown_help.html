Utilitzant Markdown

Tot i que la majoria dels textos del lloc són text pla, alguns poden necessitar format. Aquests camps, identificats amb "Suporta Markdown", admeten uns indicadors especials per donar format, el llenguatge de marques lleuger Markdown. Aquests indicadors són els següents:

Salts de línia

Per a evitar línies llargues, un salt de línia es convertirà en un espai i permetrà continuar-la a sota. Per a canviar de paràgraf, utilitza dos salts de línea.


  Aixó és
  una línia

  Però aixó és
  un altre paràgraf
{% markdown %} Aixó és una línea Però aixó és un altre paràgraf {% endmarkdown %}

Émfasi

Markdown permet cursiva i negreta amb un i dos asteriscs, respectivament. També es poden utilitzar guions baixos, però no es substituiràn en mig de una paraula.
Per mostrar un asterisc o un guió baix, s'ha d'escapar amb \


  *cursiva*
  _cursiva_
  Aquí_no_es_canvia
  _Aquí_sí_
  **negreta**
  __negreta__
  Aquí__no__es__canvia
  __Aquí__sí__

  \*Entre asteriscs\* (i una barra inclinada: \\)
{% markdown %} *cursiva* _cursiva_ Aquí_no_es_canvia _Aquí_sí_ **negreta** __negreta__ Aquí__no__es__canvia __Aquí__sí__ \*Entre asteriscs\* (i una barra inclinada: \\) {% endmarkdown %}

Subratllat i ratllat

Tot i que Markdown no té una sintaxi per al subratllat i el ratllat, pots utilitzar les etiquetes HTML corresponents, i combinarles amb sintaxi Markdown:


  <u>Subratllat</u>
  <s>Ratllat</s>
  *<u>Cursiva subratllat</u>*
  <s>**Negreta ratllat**</s>
  <s><u>Subratllat i ratllat</u></s>
{% markdown %} Subratllat Ratllat *Cursiva subratllat* **Negreta ratllat** Subratllat i ratllat {% endmarkdown %}

Capçaleres

Es poden fer capçaleres afegint d'un a sis coixinets (#), opcionalment tancades al final del títol. Cada coixinet indica un nivell menor. Els dos primers nivells també tenen una sintaxi alternativa. Com que les capçaleres de Markdown (#-######) són idéntiques a les utilitzades pel lloc, els nivell 1 i 2 poden confondre segons la pàgina on es mostrarà el text.


  # Una capçalera #
  # 1
  1 (alt)
  =======
  ## 2
  2 (alt)
  -------
  ### 3
  #### 4
  ##### 5
  ###### 6
  ####### 7 (invàlida, nivell 6 i es mostrarà un coixinet)
{% markdown %} # Una capçalera # # 1 1 (alt) ======= ## 2 2 (alt) ------- ### 3 #### 4 ##### 5 ###### 6 ####### 7 (invàlida, nivell 6 i es mostrarà un coixinet) {% endmarkdown %}

Llistes

Pots fer llistes numerades amb n. (Només importa el primer número). Per a llistes amb punts, utilitza l'asterisc (*), el guió (-) o el símbol més (+). Dues o més llistes consecutives es juntaràn en una (per això hi ha un separador entre elles).

  4. Un element
  2. Un altre
  7. ...

  ---

  * A
  * B
      * B'
      * B''
  - C
      1. C1
      2. C2
  + D

  ---

  1. Un
  2. Dos
  3. Tres
{% markdown %} 4. Un element 2. Un altre 7. ... --- * A * B * B' * B'' - C 1. C1 2. C2 + D --- 1. Un 2. Dos 3. Tres {% endmarkdown %}

Separadors

Pots insertar una línia horitzontal amb tres o més símbols (*-_) en una línia (opcionalment, amb espais entre ells).


  ### Vàlids
  ---
  - - -
  ***********

  ### Invàlids
  _*+

  +++

  a ---

  --- a
{% markdown %} ### Vàlids --- - - - *********** ### Invàlids _*+ +++ a --- --- a {% endmarkdown %}

Enllaços

Els enllaços es poden fer de dues formes: en la línea (inline) o per referència, i admeten un títol opcional mostrat al mantenir el ratolí sobre l'enllaç. Si es posa un títol, separat amb un espai de la direcció i com "Títol", 'Títol' o (Títol).


  [Exemple](http://exemple.com "Títol")

  [Exemple][1]  (La referència pot ser qualsevol valor)
  [1]: http://exemple.com

  [Exemple][]  (Referència implícita)
  [Exemple]: http://exemple.com
{% markdown %} [Exemple](http://exemple.com "Títol") [Exemple][1] (La referència pot ser qualsevol valor) [1]: http://exemple.com [Exemple][] (Referència implícita) [Exemple]: http://exemple.com {% endmarkdown %}

Imatges

Per a utilitzar una imatge penjada a internet, la sintaxi és la mateixa que als enllaços però amb una exclamació davant:


  ![Imatge](https://placehold.it/350x150?text=Imatge)
{% markdown %} ![Imatge](https://placehold.it/350x150?text=Imatge 'Títol') {% endmarkdown %}

Cites

Per a afegir una cita, afegeix > al començament de la línia:


  > Cita
  > > Cita dins d'una cita
  >
  > **Admet altres elements**
{% markdown %} > Cita > > Cita dins d'una cita > > **Admet altres elements** {% endmarkdown %}

Taules

Extenent el Markdown "estàndard", pots fer taules, amb una capçalera opcional. També pots decidir si alinear els continguts a l'esquerra (---- o :---) a la dreta (---:) o al centre (:--:). Que encaixin les barres (|) és opcional. Es poden afegir | al principi i al final, pel que aquests exemples són equivalents:


  | Capçalera    | Capçalera |  Capçalera |
  |:-------------|:---------:|-----------:|
  | A l'esquerra | Al centre | A la dreta |

  Capçalera    | Capçalera |  Capçalera
  :------------|:---------:|----------:
  A l'esquerra | Al centre | A la dreta
{% markdown %} | Capçalera | Capçalera | Capçalera | |:-------------|:---------:|-----------:| | A l'esquerra | Al centre | A la dreta | {% endmarkdown %}

Codi

Pots escriure codi o text sense format de forma que no sigui interpretat per Markdown.


  *Markdown* / `Sense *Markdown*`

  *Markdown* / ``Sense `Markdown` ``

  ```
  Sense *Markdown*
  ```

      Sense *Markdown*
{% markdown %} *Markdown* / `Sense *Markdown*` *Markdown* / ``Sense `Markdown` `` ``` Sense *Markdown* ``` Sense *Markdown* (mín. 4 espais) {% endmarkdown %}

Avançat

Comentaris

Per afegir text no visible, utilitza un comentari HTML. El comentari s'eliminarà del contingut generat.


  Text <!-- Comentari -->
{% markdown %} Text {% endmarkdown %}

HTML

Per motius de seguretat, la major part de les etiquetes es bloquejen, tret de les que es poden generar amb Markdown o excepcions com <u> i <s>. També es poden eliminar atributs o afegir-ne d'altres a les etiquetes admeses. Recomanem no utilitzar HTML fora dels casos senyalats 


  <script src="..."></script>
